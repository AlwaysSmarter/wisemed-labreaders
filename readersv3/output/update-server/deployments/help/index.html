1. Ce face aceasta aplicatie

App Update Server este consola locala prin care administrezi pachetele de update pentru aplicatiile WiseMED Reader. Aplicatia ofera catalog de aplicatii, istoric de versiuni, generare automata de release, publicare pachete si linkuri publice de download.

2. Puncte importante

  • linkurile publice se bazeaza pe prefixul public configurat
  • release-ul automat foloseste sursele locale din repo
  • pachetul de update si installerul Windows sunt artefacte separate
  • reader-ele consuma arhiva de update, nu installerul interactiv

3. Troubleshooting

  • daca lipsesc fisiere din output, reconstruieste runtime-ul din sursa curenta
  • verifica daca deployments/help/index.html exista in sursa aplicatiei
  • verifica logurile update-server pentru erori de acces la files_dir sau la repo
